리눅스 (운영체제 종류)
무료 (리누스 베네딕트 토르발스(스웨덴어: Linus Benedict Torvalds , 1969년 12월 28일 ~)는 
핀란드 헬싱키에서 태어난 스웨덴계 핀란드인으로서 소프트웨어 엔지니어이자 
리눅스 커널과 깃을 최초로 개발한 사람으로 잘 알려져 있다.)

1. 우분투 - 무료, 버전 업데이트 지속, 가장 많이 씀,
2. centOS - 버전 업데이트가 끝나 더 이상 하지 않음
3. 레드햇 - 유료, 보안이 강력, 기능들이 많이 있음
4. 코분투 - 한국에서 만들었지만 많이 안 씀

sudo - 관리자 권한
clear - 터미널 창 깨끗하게
mkdir 디렉토리명 - 디렉토리 생성
cd - 디렉토리 이동
vim 파일명 - 텍스트 파일 생성
w - 저장 
q - 나간다
wq! - 저장하고 나감 
mv 파일명 디렉토리명 - 파일 또는 디렉토리 다른 위치로 이동
mv 옛파일명 새파일명 - 파일이름 수정 
ls - 조회
