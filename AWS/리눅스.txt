운영체제 (OS) : Window, Max, Linux ...
운영체제는 소프트웨어(프로그램)이다.

회사에서는 Window or Linux 를 많이 사용하는데,
이유는 간단하다.
리눅스는 무료이기 때문에

Linux Kobuntu(무료) - 리눅스를 좋아하는 한국 사람들이 만들음
Linux *** Ubuntu(무료)
Linux CentOS(무료) - 하락세, 더이상 만들지 않음
Linux *** RedHat(유료)


리눅스 명령어를 배워보자! (tip_ tap키를 잘 활용해보자, 맥북과 명령어 똑같음!)
1. ls(List) 
   -> 현재 경로에 있는 파일 및  폴더를 보여줌 (사용 빈도율 99%)
2. cd(Change Directory) 
   -> 폴더 이동 (사용 빈도율 99%)
   cd .. (뒤로 돌아가기)
   cd ~ (처음(ROOT) 경로로 돌아감)
3. mkdir(Make Dclirectory)
    -> 폴더 만들기 (메이크 디렉토리의 줄임말)
4. clear 
    -> 명령어 깔끔하게 지워줌
5. pwd(Print Working Directory)
    -> 현재 내가 일하고 있는 폴더 경로가 어디에 있는지 확인해줌
6. vim 에디터 만들어주는 명령어
   처음에 i를 눌러야 insert 모드로 변경
   작성완료 후 esc를 누르고 
   :wq! 입력하면 최종 저장.
   vim 으로 다시 들어갈 수 있다.

   문제)dw.txt 라는 파일을 만들고 hello world 라는 내용을 저장하시오.

7.  rm (Remove)
    -> 파일 혹은 폴더를 삭제할 때 쓰는 명령어
    rm - rf (권한 무시 강제 삭제) 
    ex) rm-rf hello.txt
8. history 
   : 사용자가 사용한 명령어 내역 보여줌
9. ll
   : ls 가 단순 폴더와 파일을 확인했다면, 
     ll 은 만든사람, 날짜, 용량, 권한까지 더 상세하게 알 수 있다.

1부터 9까지 많이 사용함.


------------ 추가적 설명 ---------------

*세개씩 나누어서 읽는다.
rw-/r--/r-- : 권한
r : read (읽기) 4
w : write (쓰기) 2
x : execute (실행) 1

A : 파일소유자 권한 => rw- 
B : 그룹소유자 권한 => r--
C : 외부인(guest) 권한 => r--

문제) a.txt 
   rwxr---wx 권한을 가지고 있다
   어떤 권한을 의미하는지 서술하시오.
    - 파일소유자는 읽고, 쓰고, 실행이 가능,
      그룹소유자는 읽기만 가능, 
      외부인은 쓰기와 실행이 가능하다.

숫자로 표현했을때 777은 개발자들 사이에서 좋은 숫자가 아니다.

10. tail -f <파일이름>
     : 해당 파일을 모니터링 할 때 사용하는 명령어

ctrl + c : 탈출
